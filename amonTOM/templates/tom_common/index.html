AMON Target and Observation Manager.

How to use this website?

  • Query AMON alerts filtering by time, location, energy, resolution...
  • Create targets from these alerts or a catalog search.
  • Eventually plan and request observations of a target and follow the observation status.
  • Gather data from other observatories too and visualize them.

Permissions

  • No account is needed to see public alerts.
  • Anyone that wants to view or create targets from public alerts can sign up.
  • Each collaboration member of AMON has its own account giving access to private alerts.
    Use it if you are part of such collaboration.
